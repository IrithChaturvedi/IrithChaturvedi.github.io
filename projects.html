GitMyWiki – Cache Me If You Can (Hackathon Winner)

Summary

GitMyWiki is an AI-powered virtual assistant built at the COUNTRY Financial Hackathon 2025, where my team won 1st place out of 23 teams. The tool helps employees quickly retrieve documentation from COUNTRY’s fragmented systems (GitLab Wiki, SharePoint, Teams) by turning search into a conversational experience.

The Problem

  • Employees spent an average of 375 hours/day searching for documentation across COUNTRY Financial.
  • Information was siloed in GitLab Wiki, SharePoint, and other repositories, creating inefficiency and reduced productivity.
  • Traditional search was passive, requiring extensive digging rather than providing intelligent answers.

Technical Approach

  • Built a Retrieval-Augmented Generation (RAG) pipeline using Azure AI Search and Azure OpenAI.
  • Automated ingestion of GitLab Wiki and SharePoint data using APIs, storing content in markdown format for vector search.
  • Integrated with Azure Blob Storage for scalable document handling.
  • Frontend built with Streamlit for user-friendly, interactive query response.

Impact

  • Reduced time spent searching for documentation by providing direct, summarized answers with links.
  • Improved onboarding and training for new employees by streamlining access to resources.
  • Integrated into platforms employees already use (Teams, GitLab, SharePoint), minimizing context switching.

Future Scope

  • Enable role-based responses by connecting with HR and access management APIs.
  • Deploy GitMyWiki as a Teams app with built-in chat support.
  • Extend coverage to additional internal systems for a unified knowledge hub.

Analyzing the Real Estate Market in the US Northeast

Motivation

The Northeastern region of the USA includes Pennsylvania, Maine, New York, New Jersey, Massachusetts, Connecticut, Rhode Island, New Hampshire, and Vermont. Despite its historical and cultural significance, affordability remains a challenge for many residents.

Alternative Places in the U.S. Northeast

  • Why Stay in the Northeast? This project highlights economic stability in the Northeast while identifying affordable alternatives.
  • Focus on Affordability: Identifying cities that offer career opportunities without excessive costs.
  • Avoiding Newly Saturated Cities: Instead of moving to popular Southern cities, this project suggests sister cities with strong career prospects.
  • Strategic Move from New York: Providing data-driven insights to help individuals relocate to affordable cities without compromising economic growth.

Methods

Data Retrieval

  • Per Capita Income per US State (Kaggle)
  • Updated House Prices of Northeastern US States (Kaggle)
  • Population Statistics of US States (Data.gov)
  • Pittsburgh Home Prices Dataset (census.data.gov)

Data Processing

  • Data Cleaning: Removed null values, dropped unwanted columns.
  • Data Sorting: Grouped data by state, filtered population data for 2010-2019 trends.
  • Analysis & Visualization: Used Python Pandas, Folium, and Matplotlib to manipulate data and generate insights.

Machine Learning Model

  • Implemented Logistic Regression to predict neighborhoods based on budget, bedrooms, and school proximity.
  • Created a pipeline with StandardScaler() and LogisticRegression().
  • Optimized hyperparameters with GridSearchCV (5-fold CV).
  • Achieved 95% accuracy in predicting the best neighborhood to live in.
  • Used Zillow’s API to recommend the 10 best properties fitting user needs.

Results

  • Southern U.S. states have seen population growth due to lower costs.
  • Northeastern states, excluding New York, provide stable incomes relative to living expenses.
  • Pittsburgh, PA and Hartford, CT identified as top relocation options.

Stock Price Performance of Pharmaceutical Companies during COVID-19

Summary

This project analyzes how U.S. COVID-19 death rates correlated with stock price performance of Pfizer (PFE), AstraZeneca (AZN), and Johnson & Johnson (JNJ) between 2020–2021.

Data Sources

  • COVID-19 Data: Johns Hopkins University dataset
  • Stock Data: Alpha Vantage API (weekly adjusted close prices)

Approach

Data integration and cleaning with Python, SQL, OpenRefine, and Snakemake. Aligned pandemic milestones with stock market fluctuations to identify patterns.

Findings

  • COVID-19 peaks (Jan 2021: 300k cases/day, 4,400+ deaths/day) aligned with volatile stock responses.
  • Pfizer & AstraZeneca stocks dipped after vaccine releases due to skepticism, while Johnson & Johnson’s later rollout sustained performance.

Future Work

  • Expand analysis to Moderna & Novavax.
  • Use daily stock price data for finer-grained insights.
  • Extend analysis to global markets beyond the U.S.
