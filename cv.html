Undergraduate Researcher

Biodiversity Genomics Lab at the University of Illinois Location: Champaign, IL, USA Aug 2023 - Present In the field of computational genomics, I have designed, managed and optimized Targeted Assembly Pipelines for the Cyprinidae and Danionidae family of fish. I am also working on a publication on improving alignment accuracy with the use of statistical analysis in R and a novel weighted scoring method to handle evidences of gene duplication and hybridization. Project 1: Reconstructing Evolution and the Tree of Life in Fishes The aim of the project was to identify the genes that were targeted for sequencing in a previous experiment on the evolution of Cypriniformes - the largest group of freshwater fishes in the world. Starting with the creation of a reference set of genomic inforation for the Zebrafish (Danio Rerio) and identifying over 700 unique gene features to be used as refernece sequences for targeted assemblies in 41 species of Cyprinids. Coverage and length analysis using Python's data science libraries was conducted to assess sequences suitable for reference set. Project 2: Targeted Assembly for Reference Sequences in Cyprinidae using the CAPTUS Pipeline Using the CAPTUS pipeline developed by Dr. Edgardo Mortiz (initially to assess plant genomic data) was modified to be used to conduct targetted assembly for a family of fish (Carps and Minnows). This also includes using workflow automation softwares like NEXTflow and functional scripts using python for improving space and time complexity of the pipeline. Understanding the evolutionary relationships between species and also between genes that have duplicated, because most pipelines don't handle it. Initially only single copy genes were considered by the Stout et. al research paper on “Resolving Cypriniformes relationships”, here we try to handle duplications using the existing data we already have. I used R and its libraries, identified important characteristics and features of Amino acid, Nucleotide and Clustering alignments to determine useful and unique loci within our results. For analyzing alignment results, IQ-Tree was used to comoprehend genealogy of the family. Parallelization of IQ-Tree processes was done using Python. I also used the APE (R) package for basic unrooted tree visualizations. Identifying unique hits for each query sequence using our Extraction results and confirming if sequences overlap within the same node and range. For this a novel weighted scoring method to match best references to their respective genomic regions was developed and implemented post the extraction step using computational scripts implementing multiple data structures, searching and sorting algorithms.
